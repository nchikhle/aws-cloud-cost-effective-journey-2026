# AWS Cloud Journey 2026

## Overview

A security-first, cost-safe AWS learning journey for anyone who wants to learn AWS hands-on without the constant fear of unexpected bills.

Many beginners either delay AWS practice because they are scared of cost, or they jump directly into AWS services without understanding billing, cleanup, and account safety.

This repository follows a safer approach:

```text
Secure first.
Understand cost.
Hands-on carefully.
Monitor usage.
Clean up.
Document everything.
```

I am documenting my AWS journey step by step through **GitHub**, **LinkedIn**, and **LogiqPool**.

## The goal is to build real AWS confidence with:

```text
- AWS fundamentals
- account security
- billing and cost awareness
- hands-on practice
- troubleshooting
- cleanup habits
- interview preparation
- beginner to advanced AWS projects
- Java and Spring Boot projects on AWS
```
---

## Cost-Safe Mindset

AWS learning should be hands-on, but not careless.

Before creating any AWS resource, I should understand:

- why I am creating it
- whether it may create cost
- how it is billed
- how to monitor it
- how to clean it up
- whether a smaller or safer version is enough

Simple rule:

```text
Security first.
Cost-aware always.
Hands-on carefully.
Cleanup every time.
```
